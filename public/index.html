🚀 GenAI API - Online

API para edição de imagens e geração de vídeos usando Google GenAI

Endpoints disponíveis:

📝 Documentação

GET /api

Retorna documentação completa da API

🖼️ Editar Imagem

POST /api/edit-image

Edita ou gera imagem usando Gemini 2.5 Flash Image

🎬 Gerar Vídeo

POST /api/generate-video

Gera vídeo a partir de imagem usando Veo 3.1

🧪 Teste

GET /api/test

Endpoint de teste simples
